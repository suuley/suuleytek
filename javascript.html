Javascript

Aan idin la wadaago in yar oo ku saabsan Javascript...

SuuleyTek

Javascript waa programming language butdhig u ah teknologiyada world wide webka ayada iyo Html iyo css, wexey interactive ka dhigtaa bogagga websiteka, waxaana markeedi horeba loo sameeyey in ay ku dhexnoolato browserka, xiriirkaa oo kasbay in browserwalba uu yeesho javascript engine sida V8 Google Chrome iyo SpiderMonkey Mozilla Firefox. Saas ayeyna Javascript ku noqotay luuqadda lagu automate gareeyo bogagga websiteka. Waxyaabaha qofku markuu barto Javascript sameyn karo kuma eka website la automategareeyo ee waxuu ardaygu uga gudbi karta sameynta web app iyo mobil app sida Android app iyo Ios app adoo adeegsanayo teknolojiyadda cusub ee javascript front end frameworks. Tusaale Vue.js oo web appka lagu sameeyo iyo Vue Native oo mobil appka lagu sameeyo.
